# Perl-LWP-UserAgent-MSGraph
Perl lwp client for MSGraph API interaction

This is a subclass of LWP::UserAgent with the glue for MS Graph interaction

To install the module, run the following:

   perl Makefile.PL
   make
   make test
   make install
